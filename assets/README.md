# Assets

Ce dossier contient les assets de l'application.

## Fichiers requis

- `icon.png` - Icône de l'application (1024x1024)
- `splash.png` - Écran de démarrage (1284x2778)
- `adaptive-icon.png` - Icône adaptative Android (1024x1024)
- `favicon.png` - Favicon web (48x48)
- `notification-icon.png` - Icône de notification (96x96)

## Génération des assets

Les assets peuvent être générés avec des outils comme :
- [Expo Asset Generator](https://www.npmjs.com/package/expo-asset-generator)
- [App Icon Generator](https://www.appicon.co/)
